# IR v1 Example Specs

This folder contains one valid example for each schema:
- `dataset.spec.example.json`
- `graph.spec.example.json`
- `train.spec.example.json`
- `eval.spec.example.json`
- `training.session.spec.example.json`
- `notebook.bundle.spec.example.json`
- `runtime.handshake.spec.example.json`
- `runtime.train_event.spec.example.json`

Use these as templates for export/import and backend integration.
